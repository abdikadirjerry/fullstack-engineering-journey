Micro-School Management System
A simple, HTML-only system to view and enroll students. Built purely with raw HTML to practice structure, tables, and forms.

Features
index.html: System overview and main navigation dashboard.

students.html: Table layout showing currently enrolled students.

enroll.html: Registration form with fieldsets, text inputs, radio buttons, and selections.

about.html: Simple page detailing the project's features and goals.

How to Run
Open the project folder.

Open index.html in any web browser to view the system.
