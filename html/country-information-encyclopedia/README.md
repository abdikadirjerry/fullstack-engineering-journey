# Country Information Encyclopedia

A simple HTML-only mini project that provides basic information about countries around the world.

## Features

- Home page
- Navigation menu
- Countries information table
- Interesting facts section
- Contact page
- Feedback form
- FAQ section
- Semantic HTML5 elements
