# Counter App using Closures

## 💡 Idea

This project uses closures to create a private variable (`count`).

## 🔒 Why closure?

Because `count` cannot be accessed directly from outside.

## 🧠 Features

- increment counter
- decrement counter
- reset counter
- private state (encapsulation)

## 🚀 Learning outcome

You understand how JavaScript protects data using closures.
