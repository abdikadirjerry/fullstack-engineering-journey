# 📘 JavaScript Event Listeners

Educational resources for mastering DOM events.

## 📂 Folder Structure

- `/notes`: Explanations of `addEventListener` and common event types.
- `/examples`: Code snippets ranging from basic clicks to form handling.
- `/practice`: Unsolved and solved challenges to test your skills.
- `/mini-projects`: Three self-contained projects using event listeners.

## 🧠 Topics

- Mouse Events (`click`, `mouseenter`, `mouseleave`)
- Keyboard Events (`keydown`)
- Input & Form Events (`input`, `submit`)
- The Event Object & `preventDefault()`

## 🚀 Projects

1. **Live Preview:** Real-time text updating.
2. **Counter App:** Managing state with button clicks.
3. **Random Color Generator:** Dynamic CSS manipulation.
