# 📘 JavaScript Spread & Rest Operators

This folder is part of my **Web Dev Journey** repository. It focuses on the versatile `...` syntax, distinguishing between its use for **expanding** data (Spread) and **collecting** data (Rest).

---

## 📂 Folder Structure

```text
spread-rest/
├── notes/             # Expansion vs. Collection logic
├── examples/          # Functional demos (Easy, Int, Adv)
├── practice/          # Solutions for array/object manipulation
└── mini-project/      # Task Manager with Immutable Updates
🧠 Topics Covered
Spread Operator: Cloning and merging without mutating original data.

Rest Parameters: Creating flexible functions that accept dynamic arguments.

Object Destructuring: Isolating specific keys while grouping the remainder.

Immutable State: Using spread to update arrays and objects in a "React-friendly" way.

🚀 Mini Project: Task Manager
Located in mini-project/task-manager.js.

This project demonstrates Immutable State Updates, where the addTask and toggleTask methods use the spread operator to ensure data is updated safely without direct mutation of the original array.

📌 Key Takeaway
"Spread pours the data out; Rest gathers it up."
```
