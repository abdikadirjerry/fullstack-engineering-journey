# Interactive Kanban Board

A modular task management application demonstrating complex DOM event manipulation and physical UI state synchronization.

## Technical Milestones

- **HTML5 Drag & Drop API:** Implemented `dragstart`, `dragover`, `dragleave`, and `drop` event listeners.
- **Data Transfer Object:** Utilized `e.dataTransfer` to pass component IDs securely between distinct UI nodes during drag operations.
- **State Synchronization:** Ensured visual column changes instantly mutate the underlying JSON array logic and persist to LocalStorage.
