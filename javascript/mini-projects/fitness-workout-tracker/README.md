# FitTrack | Client-Side Workout Logger

A lightweight, high-performance web application designed to locally track, calculate, and analyze workout volumes and performance progressions.

## Technical Architecture

The application is engineered using clean JavaScript separating structural models, UI view representations, state tracking, and local operations.

- **Vanilla JS Foundations:** Decoupled functional logic avoiding heavy abstraction or complex toolchains.
- **LocalStorage Interface:** Seamless data durability over sessions without remote network requirements.
- **Dynamic Metric Pipelines:** Built-in programmatic estimation updates containing tracking modules (`1RM via Epley Formula`).
