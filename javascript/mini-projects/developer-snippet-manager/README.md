# Developer Snippet Manager

A JavaScript application for storing and organizing reusable code snippets.

## Features

- Save code snippets
- Language categories
- Search snippets
- Filter by language
- Favorite snippets
- Delete snippets
- Statistics dashboard
- Local Storage persistence

## Languages

- JavaScript
- HTML
- CSS
- Python
- SQL

## Technologies

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage
