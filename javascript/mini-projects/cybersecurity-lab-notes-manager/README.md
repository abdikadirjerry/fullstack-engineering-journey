# 🔐 Cybersecurity Lab Notes Manager

A JavaScript application for organizing cybersecurity lab notes, useful commands, and penetration testing write-ups.

---

## Features

- Add cybersecurity lab notes
- Save Linux, Networking, Web Security, Cryptography and Forensics notes
- Store useful terminal commands
- Search notes instantly
- Filter notes by category
- Delete notes
- Dashboard statistics
- Local Storage persistence
- Responsive design

---

## Technologies

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API

## How to Run

1. Clone or download the project.
2. Open `index.html` in your browser.
3. Add your first lab note.
4. Search or filter notes by category.
5. All data is automatically saved in Local Storage.
