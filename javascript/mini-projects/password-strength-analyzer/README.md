# Password Strength Analyzer Pro

A professional JavaScript project that analyzes password security and generates strong passwords.

## Features

- Real-time password analysis
- Password generator
- Copy password
- Show / Hide password
- Entropy calculation
- Strength score
- Progress bar
- Requirement checklist

## Technologies

- HTML
- CSS
- JavaScript

## Concepts Practiced

- DOM Manipulation
- Event Listeners
- Functions
- Objects
- Regular Expressions
- Password Validation
- UI Updates

Open index.html in your browser.
