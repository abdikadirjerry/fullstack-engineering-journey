# Student Management System

A practical vanilla JavaScript application designed to execute isolated CRUD operations via a modular architecture.

## Technical Scope

- **Data Flow:** Form Input -> `students.js` State Mutation -> `storage.js` Persistence -> `dom.js` Table Render.
- **Security:** Implements basic HTML sanitization on user inputs to prevent script injections.
- **Architecture:** Strictly adheres to ES Module separation of concerns.

> Developed for abdikadir.dev portfolio milestones.
