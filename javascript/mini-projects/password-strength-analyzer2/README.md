# 🔐 Password Strength Analyzer

A JavaScript mini project that evaluates password strength in real time and helps users create stronger passwords.

---

## Features

- Real-time password analysis
- Password strength meter
- Password requirements checklist
- Show / Hide password
- Random secure password generator
- Copy password to clipboard
- Responsive design

---

## Technologies

- HTML5
- CSS3
- JavaScript (ES6)

## How to Run

1. Clone or download the project.
2. Open `index.html` in your browser.
3. Type a password or generate one.
4. View the strength meter and requirements.
5. Copy the generated password if needed.

---

## Password Rules

- Minimum 8 characters
- At least one uppercase letter
- At least one lowercase letter
- At least one number
- At least one special character
