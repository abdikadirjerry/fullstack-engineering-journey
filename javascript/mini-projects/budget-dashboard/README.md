# 💰 Personal Budget Dashboard

A professional JavaScript mini project for tracking income, expenses, and account balance.

---

## Features

- Add income transactions
- Add expense transactions
- Delete transactions
- Income summary
- Expense summary
- Balance calculation
- Transaction filtering
- Local storage persistence
- Responsive design

---

- HTML5
- CSS3
- JavaScript (ES6)
