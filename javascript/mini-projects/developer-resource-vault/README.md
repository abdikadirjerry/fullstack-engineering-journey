# Developer Resource Vault

A JavaScript application for saving and organizing learning resources.

## Features

- Save useful resources
- Store title and URL
- Categorize resources
- Search resources
- Filter by category
- Delete resources
- Local Storage persistence
- Statistics dashboard

## Categories

- Frontend
- Backend
- Database
- DevOps
- Cybersecurity

## Technologies

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage

## How To Run

1. Open index.html
2. Add resources
3. Search and filter resources
4. Data is automatically saved
