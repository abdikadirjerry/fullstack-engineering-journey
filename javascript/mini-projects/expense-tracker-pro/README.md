# 💰 Expense Tracker Pro

A simple JavaScript expense tracker application built using:

- HTML
- CSS
- JavaScript
- LocalStorage

## 🚀 Features

- Add expenses
- Delete expenses
- Search expenses
- Calculate total expenses
- Save data using LocalStorage
- Dynamic DOM rendering
- Modular JavaScript structure
