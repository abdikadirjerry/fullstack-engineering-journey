# 🎬 Movie WatchList

A JavaScript movie watchList application for managing movies you want to watch.

Built using:

- HTML
- CSS
- JavaScript
- LocalStorage

## 🚀 Features

- Add movies
- Delete movies
- Search movies
- Save movies using LocalStorage
- Dynamic movie rendering
- OOP with JavaScript classes
