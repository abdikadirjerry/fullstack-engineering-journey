# Job Application Tracker

A professional JavaScript project for tracking job applications.

---

## Features

- Add applications
- Delete applications
- Search companies
- Filter by status
- Statistics dashboard
- Dark mode
- Export data
- Local storage

---

## Status Types

- Applied
- Interview
- Rejected
- Accepted
